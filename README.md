# 💊 MedConnect Application

MedConnect é uma aplicação web/mobile que conecta pacientes que precisam de medicamentos manipulados a farmácias de manipulação de forma rápida, segura e eficiente.

O objetivo da plataforma é digitalizar o processo de envio de receitas, solicitação de orçamentos e acompanhamento de pedidos, promovendo praticidade para o usuário e maior alcance para as farmácias.

---

## 🚀 Problema

Pacientes que precisam de medicamentos manipulados geralmente enfrentam:

* Dificuldade para comparar preços
* Necessidade de ir presencialmente até várias farmácias
* Processo manual e demorado para envio de receitas
* Falta de transparência nos prazos

Farmácias de manipulação, por outro lado, muitas vezes possuem baixa presença digital e dificuldade para captar novos clientes online.

---

## 💡 Solução

O MedConnect atua como uma plataforma intermediadora que permite:

### 👤 Para o Paciente

* Criar conta e autenticar-se
* Enviar receita médica (imagem ou PDF)
* Solicitar cotação
* Receber propostas de diferentes farmácias
* Comparar preço e prazo
* Acompanhar status do pedido
* Avaliar o serviço

### 🏪 Para a Farmácia

* Criar perfil empresarial
* Receber solicitações de orçamento
* Enviar valores e prazos
* Gerenciar pedidos
* Atualizar status
* Receber avaliações
